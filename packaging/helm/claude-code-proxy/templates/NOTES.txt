Important Next Steps:

1. Add Claude Accounts:
   The proxy needs Claude accounts to function. You have two options:

   a) Use the Web UI to add accounts:
      - Access the application URL above
      - Navigate to the web interface
      - Use the OAuth flow to add Claude accounts

   b) Manually create accounts.json on the persistent volume:
      - Execute into the pod: kubectl exec -it {{ include "claude-code-proxy.fullname" . }}-<pod-id> -- /bin/bash
      - Create /data/accounts.json with your OAuth credentials
      - Restart the pod or wait for hot reload

2. Check Health Status:
   kubectl exec -it {{ include "claude-code-proxy.fullname" . }}-<pod-id> -- curl http://localhost:8000/health

3. View Logs:
   kubectl logs -f deployment/{{ include "claude-code-proxy.fullname" . }}

4. Configure Your Claude SDK Client:
   Set these environment variables in your client application:

   export ANTHROPIC_BASE_URL=http://<service-url>:{{ .Values.service.port }}
   export ANTHROPIC_API_KEY=<get-from-web-ui>
