Claude Code Proxy

Multi-account Claude API proxy with automatic rotation

Multi-Account Rotation

Automatically rotate between multiple Claude accounts to maximize throughput and avoid rate limits.

API Compatible

Drop-in replacement for the Anthropic API. Works with existing SDKs and tools.

Web UI

Manage accounts, monitor usage, and configure settings through a built-in web interface.
