Claude Proxy Multi

Multi-account Claude Code proxy with automatic rotation

Docker (Recommended)

curl -fsSL https://joachimbrindeau.github.io/claude-proxy-multi/install.sh | bash

Homebrew

brew tap joachimbrindeau/claude-code-proxy
brew install claude-code-proxy

Helm

helm repo add ccproxy https://joachimbrindeau.github.io/claude-proxy-multi/charts
helm install ccproxy ccproxy/claude-code-proxy
